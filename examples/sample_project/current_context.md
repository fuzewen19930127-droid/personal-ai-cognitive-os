# Current Context

## Current State

The sample project is a sanitized public template for a local-file AI cognitive memory system.

It contains:

- control rules
- project memory
- immediate task memory
- error routes
- external knowledge manifest
- real-time verification layer
- controlled evolution log
- static dashboard generator
- validation and sensitive-content scan scripts

## Active Assumptions

- The public repository should stay local-file-first.
- Markdown and JSON are enough for the first version.
- External knowledge is evidence, not automatic policy.
- High-change facts still need current verification.
- No secrets or private raw data belong in the public template.
