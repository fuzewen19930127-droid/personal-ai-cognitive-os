# Release Notes

## v0.1.0

Initial public MVP.

Included:

- local-file AI cognitive memory structure
- three-layer recall model
- rule cards
- project card template
- immediate task memory
- error routes
- external knowledge manifest
- real-time verification layer
- controlled evolution log
- static dashboard generator
- repository validator
- sensitive-content scanner
- sanitized sample data
- proof case for error-route to rule evolution

Known limits:

- no database or vector search
- no hosted dashboard
- no automatic external knowledge ingestion
- no built-in multi-agent runtime
- screenshot or GIF assets may be added in a later release
