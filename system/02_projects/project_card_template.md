# Project Card Template

Use this template for long-running projects.

```yaml
project_id:
display_name:
system_id:
status:
owner:
entry_path:
created_at:
updated_at:
sensitivity:
```

## Goal

Describe what the project is meant to achieve.

## Scope

State what is included and what is intentionally excluded.

## Current Context

Summarize the current project state.

## Files

| File | Purpose |
|---|---|
| `project_brief.md` | Goal, scope, owner, entry points |
| `current_context.md` | Current state and active assumptions |
| `progress_log.md` | Timeline and decisions |
| `evidence_trace.md` | Validation, source traces, outputs |
| `acceptance_checklist.md` | Completion criteria |

## Next Action

Write the next concrete action.

## Verification

List commands, checks, screenshots, or review steps used to validate the project.
