# Project Layer

Long-running work needs project cards.

Project cards are the main memory for a specific project. They prevent project state from living only in chat context.

## Minimal Project Card

```text
project_brief.md
current_context.md
progress_log.md
evidence_trace.md
acceptance_checklist.md
```

Copy `examples/sample_project` as a starting point.

## Completion Criteria

- Goal is completed or explicitly downgraded.
- Deliverable paths are clear.
- Verification is recorded.
- Remaining risks are listed.
- Next action is explicit.
